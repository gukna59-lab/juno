# Juno

Адаптивный мессенджер в стиле Telegram: регистрация, вход, редактирование профиля, аватарки, просмотр чужого профиля, поиск друзей по юзу, чаты, сообщения, подтверждение фото с подписью, запись голосовых через микрофон и интерфейс звонков с доступом к микрофону/камере.

## Запуск на ПК

Открой `index.html` в браузере или запусти локальный сервер из этой папки:

```bash
python -m http.server 4173
```

После этого сайт будет доступен на ПК:

```text
http://127.0.0.1:4173
```

## Запуск на телефоне в одной Wi-Fi сети

1. Запусти сервер на ПК.
2. Узнай IP-адрес ПК в локальной сети.
3. Открой на телефоне:

```text
http://IP_ТВОЕГО_ПК:4173
```

## Бесплатная ссылка для других людей

Самый простой вариант без оплаты:

1. Создай репозиторий на GitHub.
2. Загрузи туда `index.html`, `styles.css`, `app.js`, `README.md`.
3. Включи GitHub Pages в настройках репозитория.

GitHub даст публичную ссылку. Но сейчас данные хранятся локально в браузере, поэтому у каждого человека будет своя отдельная демо-база.

## Что нужно для настоящего общего мессенджера

Для реальных аккаунтов между разными людьми, общих сообщений, настоящих звонков и синхронизации нужен бесплатный backend, например Supabase или Firebase. Для звонков дополнительно нужен WebRTC-сигналинг: сам браузер умеет передавать звук/видео, но пользователям надо где-то обменяться данными подключения.

Микрофон и камера работают на `localhost` или на опубликованном HTTPS-сайте. На телефоне через обычный `http://IP_ПК:4173` браузер может заблокировать запись и звонки из-за политики безопасности.
